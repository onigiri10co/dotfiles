# roles/rosetta
Rosetta 2 を使えば、Apple シリコンを搭載した Mac でも、Intel プロセッサ搭載 Mac 用に開発された App を使えるようになります。



## Dependencies
- nothing



## Usage
起動したアプリがインテル対応の場合、自動的に使用されるため、インストールさえしておけば特に意識することはない。
