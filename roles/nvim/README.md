# roles/nvim
neovim/neovim: Vim-fork focused on extensibility and usability



## Dependencies
- homebrew



## Usage
- `nvim` で起動できる（alias vim=nvim を設定している）。
- 起動時に、`$XDG_CONFIG_HOME/init.lua` を読み込んで起動する。
