## Installation
インストール後、以下の Cheat Sheets リポジトリを配置する。
Private リポジトリであり、セットアップ時はアクセスできないので手動で実施する。

- [onigomex/cheatsheets: My cheat sheets](https://github.com/onigomex/cheatsheets)

```
% ghq get git@github.com:onigomex/cheatsheets.git
```
